$ UdpChat <mode> <command-line arguments> : 
Start the program for server or client, 
(for example : UdpChat -c for client and UdpChat -s for server). 
The server mode takes one argument, its listening port. 
The client mode should take four arguments: the client’s nick-name, 
the server ip address, the server’s listening port number, and this 
client’s port number for listening.
